# Ruphus ERP — Landing Page

Site institucional do Ruphus ERP, com 22 módulos integrados para gestão empresarial completa.

## Stack

- HTML/CSS/JS puro (zero dependências)
- Fontes: Fraunces + Outfit + JetBrains Mono
- Dark mode nativo
- PWA-ready

## Seções

- **Hero** com typing effect e CTA para demonstração
- **Trust badges** reais (SSL/HSTS, Supabase RLS, LGPD, Vercel Edge, Sentry)
- **Bento grid** com mockup interativo do sistema (3 tabs)
- **Antes/Depois** — comparativo visual
- **22 Módulos** organizados em 4 categorias com tabs interativas
- **Ecossistema** — grid com todos os módulos e status
- **Calculadora ROI** — interativa com inputs dinâmicos
- **Recursos & Insights** — hub de conteúdo com filtros por categoria
- **FAQ** — 12 perguntas em 4 categorias com accordion e filtros
- **CTA Demo** — formulário de lead capture com validação
- **WhatsApp FAB** + modal de agendamento

## SEO

- Open Graph + meta tags otimizadas
- JSON-LD Schema (SoftwareApplication + FAQPage com 12 Q&As)
- Semântica HTML5 completa

## Deploy

Arquivo único `index.html` — pode ser servido como rota estática no Next.js 15 ou via qualquer CDN.
